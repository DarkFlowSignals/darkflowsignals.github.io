Spot Row Variants — pick one

Five ways to display the spot price + change. Each shown in a real card context (CRWV runner @ Day 24). Bottom callouts show how each handles edge cases.

Variant A — Today's % only (current v16)
What's the underlying doing right now.
$CRWV Bullish Primary
Spot $124.50 ▲+2.1% today CoreWeave
Pros: Live tape pulse, dashboard feels alive.
Cons: For day-24 positions, today's ±2% is noise. Doesn't tell trade story.
Variant B — Since-entry only
Has the underlying actually moved with us?
$CRWV Bullish Primary
Spot $124.50 ▲+38% since 4/9 CoreWeave
Pros: Trade narrative. Members see whether option run came from real stock movement vs IV/theta.
Cons: No live pulse. Day-1 positions show ~0% which feels uninformative.
Variant C — Both, since-entry leads (recommended)
Trade story first, live pulse second.
$CRWV Bullish Primary
Spot $124.50 ▲+38%since 4/9 · ▲+2.1%today · CoreWeave
Pros: Both data points visible. Since-entry leads (trade narrative), today is secondary (live pulse). Color hierarchy: since-entry is brighter green, today is softer green-soft.
Reads as: "Stock is up 38% since we got in (real move), and another 2.1% today (still rallying)."
Variant D — Both, today leads
Live pulse first, trade story second.
$CRWV Bullish Primary
Spot $124.50 ▲+2.1%today · ▲+38%since 4/9 · CoreWeave
Pros: Today is the most "live" feel — leading with it makes the dashboard feel real-time.
Cons: Trade narrative gets demoted. Today's noise is louder than the durable signal.
Variant E — Stacked two lines
Vertical hierarchy.
$CRWV Bullish Primary
Spot $124.50 ▲+38% since 4/9 CoreWeave
Today ▲+2.1% · $122.04 → $124.50
Pros: Dedicated row for today's intraday move. Most informative.
Cons: Adds a row. Spot row no longer fits in one line, card grows.
