Methodology

Three layers. One conviction score.

Every signal runs through three independent checks before it hits your feed. If the layers don't agree, you don't see it.

01 / SIGNAL

Institutional Flow Detection

We scan thousands of institutional trades every day looking for unusual size and positioning. When the same ticker lights up across multiple independent filters at the same time, that's not noise. That's the signal.

Primary signal layer
02 / CHART

Technical Confirmation

A flow signal without chart confirmation is just noise. Every detection gets cross-checked across multiple timeframes. If the technicals don't back it up, we pass. Simple as that.

Confirmation layer
03 / MACRO

Catalyst Context

Big flow ahead of CPI day hits different than flow on a random Tuesday. We check the macro calendar, earnings risk, and sector rotation to make sure the trade has a reason to work right now.

Context layer
CONVERGENCE BONUS

When multiple independent data sources stack up on the same ticker, the conviction score goes up. More agreement = stronger signal.
